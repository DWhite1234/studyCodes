```
pwd		:显示当前目录的绝对路径

ls		:列出目录内容
    -a	:列出全部文件,包含隐藏文件
	-l	:长数据串列出,包含文件的属性与权限等数据
	
cd		:切换目录

mkdir	:创建新的目录
	-p	:创建多层目录
	eg	:mkdir -p aa/bb
	
rmdir	:删除一个空目录

touch	:创建一个空的文件

cp		:复制文件
	eg	:cp aa.sh ss/
	-r	:复制整个文件夹
	eg	:cp -r aa ss/
	
rm		:删除文件或目录
	-r	:删除所有文件包括目录(需要确认)
	-f	:强制删除,不需要确认
	-v	:显示删除过程
	
mv		:移动文件或目录,或者重命名文件
	eg	:mv test.txt  ttt.sh//重命名
	eg	:mv a.sh ./dd   //移动文件
	eg	:mv aa bb/		//移动目录
	
cat		:输出文件内容
	-n	:包括行号,与空行
	
more	:文件分屏查看器(整体加载)
	space:向下翻页
	enter:向下一行
	q	:离开
	ctrl+f:向下一屏
	ctrl+b:返回上一屏
	=	:显示当前行号
	:f	:显示当前文件名和行号
	
less	:文件分屏查看器(按需加载)
    space:向下翻动一页；
    pagedown:向下翻动一页
    pageup	:向上翻动一页；
    /字串	:向下搜寻『字串』的功能；n：向下查找；N：向上查找；
    ?字串	:向上搜寻『字串』的功能；n：向上查找；N：向下查找；
    q  	:离开 less 这个程序；

echo	:输出内容到控制台
	-e	:转义输出
	
head	:显示文件前10行内容
	-n	:指定显示的行数
	
tail	:默认输出文件后10行内容
	-n	:指定输出的行数
	-f	:显示文件最新追加的内容,监视文件的变化
	
>		:覆盖写(将能显示到控制台的内容写进文件)
	eg	:head -n 5 > ttt.sh
>>		:追加写(将能显示到控制台的内容写进文件)
	eg	:head -n 1 >> ttt.sh
	
ln -s 	:软链接(类似window的快捷方式)
	ln -s 文件名/目录名  快捷方式名
	rm -rf 快捷方式名  删除快捷方式
	
history	:查询已经历史命令
```
