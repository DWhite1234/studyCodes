# 一般模式
```txt
	1.yy复制一行
	2.yny复制n行
	3.yw复制一个单词
	4.p粘贴
	5.dd删除一行
	6.dnd删除n行
	7.dw删除一个单词
	8.u撤销
	9.G跳到最后一行
	10.1+G跳到首行
	11.n+G跳到第n行
	12.^跳到行头
	13.$跳到行尾
```

# 2.编辑模式
```txt
	1.i 当前光标前插入
	2.I 当前光标行首插入
	3.a 当前光标后一位插入
	4.A 当前光标行尾插入
	5.o 下一行插入
	6.O 上一行插入
```

# 3.命令模式
```txt
	以:开头
		1.:w 保存
		2.:q 退出
		3.:! 强制执行
		4.:wq 保存后退出
		5.:q! 不保存退出
		6.:set nu 开启行号
		7.:set nonu 关闭行号
		8.:noh 关闭高亮
		9.%s/需要替换的内容/替换的内容[/g] 默认每行只替换一次[全局替换]	
	以/开头	
		1./需要查找的内容 会黄色高亮显示
```
