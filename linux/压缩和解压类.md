```shell
1．基本语法
tar  [选项]  XXX.tar.gz  将要打包进去的内容  （功能描述:打包目录,压缩后的文件格式.tar.gz）

2.选项说明
-z	打包同时压缩
-c	产生.tar打包文件
-v	显示详细信息
-f	指定压缩后的文件名
-x	解包.tar文件
-C  指定解压目录(大写C)

3.使用

打包多文件
	tar -zcvf 打包后的文件名.tar.gz  要打包的文件1.txt 要打包的文件2.txt
打包目录
	tar -zcvf 打包后的文件名.tar.gz  
解压(只能解压到当前目录)
	tar -zxvf 打包后的文件名.tar.gz
解压到指定目录
	tar -vxf 打包后的文件名.tar.gz	 -C 指定目录
	
```

```shell
gzip/gunzip

1．基本语法
gzip 文件		（功能描述：压缩文件，只能将文件压缩为*.gz文件）
gunzip 文件.gz	（功能描述：解压缩文件命令,只能解压到当前目录）

2．经验技巧
（1）只能压缩文件不能压缩目录
（2）不保留原来的文件

3.案例
	gzip 需要打包的文件
	gunzip 需要解压的文件.gz
```

```shell
zip/unzip

1．基本语法
zip  [选项] XXX.zip  将要压缩的内容 		（功能描述：压缩文件和目录的命令）
unzip [选项] XXX.zip	

2.选项说明
zip -r	压缩目录
unzip -d 指定解压后文件的存放目录

3.案例
	zip 压缩的文件.zip  要压缩的文件1.txt 要压缩的文件2.txt
	zip 压缩的文件.zip -r 目录
	unzip  解压的文件.zip -d 目录
```
