# 数据类型
INTEGER                 integer
UNSIGNED_INT            integer
BIGINT                  long
UNSIGNED_LONG           long
TINYINT                 byte
UNSIGNED_TINYINT        byte
SMALLINT                short
UNSIGNED_SMALLINT       short


FLOAT                   float
UNSIGNED_FLOAT          float   
DOUBLE                  double
UNSIGNED_DOUBLE         double
DECIMAL                 bigdecimal

BOOLEAN                 boolean
VARCHAR(不可以指定位数)  varchar
CHAR(可以指定位数)       varchar

注意点:
    1.会自动转换成大写,如果一定要用小写用""引起来
    2.只支持单表查询,不支持多表查询
    3.支持别名
