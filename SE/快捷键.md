# IDE 快捷键
ctrl + shift + Enter  如果没有分号,则会补全分号;如果有分号,则会自动换行
alt + Enter           自动补全代码
ctrl+shift+v          粘贴板
Ctrl + Q              查看JavaDoc
Ctrl+F12              查看当前类所有方法
Ctrl + Shift + Alt + T  代码重构
CTRL+E                最近更改的代码
Ctrl+Shift+上下键      上下移动代码
Alt+1                 快速打开或隐藏工程面板
SHIFT+ALT+INSERT      快速新建文件
ctrl + shift + U      将字母变为 大小写
Space                 继承树中新增类
